# chapanda-style-preset
chapanda style preset

## 基础层
* base 提供基础的 scss 预设、颜色
* unocss 提供基础的 unocss 预设 + scss 字体
* tailwind 提供基础的 tailwindcss 预设 + scss 字体

## 服务层
* antd 提供 antd 主题预设  + scss 颜色
* ep 提供 element 主题预设  + scss 颜色
* vant 提供 vant 主题预设  + scss 颜色

## 应用层
* preset 提供全部功能支持
